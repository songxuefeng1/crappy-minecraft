  - 1. install VS Code *(if you don't have)*
  - 2. install python3 plugin in VS Code
  - 3. open the terminal in the VS Code
  - 4. commands *(see down, for Windows, command need to run one by one!)*
       ```{bash}
       python3.exe -m venv .venv
       cd .venv/Scripts
       ./activate
       pip install ursina
       ```
       commands *(for Mac OS)*
       ```{bash}
       python3 -m venv venv
       . venv/bin/activate.fish
       pip3 install ursina
       ```
